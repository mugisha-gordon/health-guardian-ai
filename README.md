## ðŸš€ Getting Started

Follow these steps to run the application locally:

```bash
# Clone the repository
git clone https://github.com/mugisha-gordon/health-guardian-ai.git
cd health-guardian-ai

# Install dependencies
npm install

# Start development server with hot reload
npm run dev
```

To create a production build:

```bash
npm run build
```

The compiled output will be in `dist/` and ready for deployment on:
- GitHub Pages
- Vercel
- Netlify
- Any static hosting provider
